# 中杯 客户端



## 开始步骤

### 先决条件

云服务器需要安装配置的内容。

下载安装：

*   Java 建议在 java21 或以上版本运行
```sh
java -version
java 21.0.8 2025-07-15 LTS
``````
*  maven 建议在 maven3.9.11 以上运行
```sh
mvn -version
Apache Maven 3.9.11 
``````

### 启动步骤

1. 将midcup_chat项目导入IDEA,运行Main.java

2. windows版本双击运行已打包文件中的midcup.exe也可运行
